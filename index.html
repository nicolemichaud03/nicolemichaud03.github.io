Recipe Recommendation System with User Dietary Preference Information

From over 226,000 users and their recipe ratings on food.com, Natural Language Processing (NLP) techniques were employed to categorize users as different diet types. A multi-task Neural Network model created with Tensorflow Recommenders used text processing to incorporate the diet-types of users while retrieving and ranking the top 3 recipe recommendations for each user. The recommendation retrieval was then optimized using Scalable Nearest Neighbors (ScaNN).
